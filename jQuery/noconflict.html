jquery - Metodo noConflict

En ocaciones se requiere la utilizacion de otros frameworks en una pagina. Pueden existir problemas con ellos pues utilizan el simbolo $ como atajo.

Si dos frameworks utilizan el mismo atajo, uno de ellos se detendra

El metodo noConflict() "suelta" el identificador $, con el fin que otros scripts puedan utilizarlo
