JavaScript Arrays (Busquedas)

IndexOf()

Devuelve el indice donde se encuentra el elemento especificado.

lastIndexOf()

Devuelve el indice donde se encuentra el ultimo elemento especificado

includes()

Devuelve True si se encuentra el elemento y False si no existe en el arreglo

find()

Devuelve el primer valor que cumple con una funcion de prueba

findIndex()

Devuelve el indice del primer valor que cumple con una funcion de prueba

findLast()

Recorre el arreglo y devuelve el valor del primer elemento que satisface una funcion.

findLastIndex()

Recorre el arreglo y devuelve el valor del indice del primer elemento que satisface una funcion.
