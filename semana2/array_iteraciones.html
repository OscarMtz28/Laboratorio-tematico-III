JavaScript Arrays (Iteraciones)

Realiza un instruccion por cada elemento del arreglo

forEach()

map()

El metodo map() crea un nuevo arreglo al ejecutar una funcion en cada elemento del arrreglo

filtra()

Crea un nuevo arreglo unicamente con los elementos que cumplan una condicion.

reduce()

Ejecuta una funcion en cada elemento del arreglo para producir un valor final

every()

Comprueba si todos los elementos de un arreglo cumplen una condicion

some()

Comprueba si alguno de los elementos de un arreglo cumple con una condicion

some()

Regresa un arrego desde cualquier objeto que cuente con la propiedad lenght o desde cualquier objeto iterable

entries()

Genera un objeto que contiene pares de indices y valores
