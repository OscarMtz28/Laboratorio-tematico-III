Cadenas en JavaScript

Se puede definir una cadena utilizando comillas simples o dobles.

Longitud de cadenas

Se puede obtener la longitud o tama√±o de una cadena a traves de la propiedad lenght

charArt y charCodeArt

El metodo charArt devuelve el caracter que se encientra en una posicion especifica (indice) dentro de una cadena. El metodo charCodeArt devuelve el codigo UTF-16 del caracter

Metodo at

El metodo at() devuelve el caracter que se encuentra en un indice especifico. La diferencia con charAt es que se pueden usar indices negativos (de derecha a izquierda)

Convertir a mayusculas y minusculas

Papara convertir a mayusculas se usa el metodo toUpperCase(). Para convertir en minusculas se utiliza toLowerCase()

Concatenar

Se puede concatenar con el operador +, pero tambien existe el metodo concat() que permite unir cadenas

Trim

El metodo trim() elimina los espacios en blanco al inicio y al final de una cadena

Repetir

El metodo repeat() devuelve una cadena con un numero especificaso de repeticiones de un string

Extraer texto

Se pueden utilizar 3 metodos para extraer una parte de una cadena: slice(inicio, fin), substring(inicio, fin) y substr(inicio,fin)

Padding

Los metodos padStart() y padEnd() permiten rellenar una cadena con los caracteres especificados

Split

Si se tiene una cadena con elementos separados por un caracter especifico se pueden convertir en un arreglo a traves de split()

Replace

Este metodo permite encontrar una cadena especifica y reemplazarla por otra
